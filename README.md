# 📖 About Rivique Beauty

**Rivique Beauty** is an AI-powered bridal beauty and salon booking platform that helps users discover, compare, and book trusted salons, makeup artists, hairstylists, and beauty professionals across Delhi NCR.

The platform combines intelligent recommendations, seamless appointment booking, vendor management tools, and a premium luxury user experience to simplify the bridal beauty journey while empowering beauty professionals to grow their businesses digitally.

---

# 🎯 Problem Statement

## Challenges for Customers

* Finding trusted bridal beauty professionals is difficult.
* Comparing services, pricing, and reviews takes significant time.
* Booking appointments often requires multiple calls and follow-ups.
* Brides struggle to find professionals matching their style and budget.

## Challenges for Salon Owners & Beauty Professionals

* Limited online visibility.
* Difficulty reaching qualified customers.
* Manual appointment management.
* Lack of centralized business tools.

# 🚀 Getting Started

## Clone Repository

```bash
git clone <repository-url>
cd rivique-beauty
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create:

```env
.env.local
```

Add:

```env
DATABASE_URL=
NEXT_PUBLIC_API_URL=
JWT_SECRET=
GEMINI_API_KEY=
```

## Generate Prisma Client

```bash
npx prisma generate
```

## Run Database Migration

```bash
npx prisma migrate dev
```

## Start Development Server

```bash
npm run dev
```

Application runs on:

```bash
http://localhost:3000
```
